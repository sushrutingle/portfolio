# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this portfolio website, please send an email to sushrutingle@gmail.com.

Please include the following information:
- Type of vulnerability
- Full paths of source file(s) related to the vulnerability
- Location of the affected source code
- Any special configuration required to reproduce the issue
- Step-by-step instructions to reproduce the issue
- Proof-of-concept or exploit code (if possible)
- Impact of the issue

We will acknowledge your email within 48 hours and provide a more detailed response within 7 days, indicating the next steps in handling your report.

## Response Timeline

- **Initial Response**: Within 48 hours
- **Detailed Response**: Within 7 days
- **Resolution**: Depending on severity and complexity

## Scope

This security policy applies to:
- The portfolio website code
- Associated JavaScript and CSS files
- Build and deployment processes

Out of scope:
- Third-party services (Google Fonts, etc.)
- User-generated content (none on this static site)
